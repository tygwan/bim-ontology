Help: Overview

Select a TTL data source and click Load to set the active RDF graph. All stats and charts update from the active graph.

  • Total Triples: total RDF statements in the graph.
  • Objects: BIM elements/instances (physical objects).
  • Property Values: literal values attached to elements and property sets.
  • Categories: distinct category labels.
  • Max Depth: deepest hierarchy level from navis:maxHierarchyLevel.

Help: Hierarchy

System Path Tree (top-right): Collapsible tree from sp3d:hasSystemPath. Shows Project > Area > Unit > System. Click nodes to expand/collapse.

Navisworks Hierarchy Tree (below): Miller Columns drill-down using navis:hasParent. Click a node to see its children in the next column. Scroll horizontally for deeper levels.

  • Depth selector: Set how deep to load. Lower = faster. Start with 3-4.
  • Show Graph: Canvas visualization of selected node and children.
  • Property Summary: Appears when you select a node. Shows category breakdown and status distribution of all descendants.
  • Hierarchy Details: Shows children table for the selected node.

Tip: Use navis-via-csv-v3.ttl for PropertyValue aggregation data.
